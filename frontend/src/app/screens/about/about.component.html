About

The Hippodrome is a online Horse Racing sports management game where you take control of a Horse Racing team.
It's a free game with 8-bit look and feel with real-time 2D races.

Features

  • Manage your Horse Stamina, Form, etc, before each Race
  • Real-time 2D race system
  • Buy specific trainers for each Skill of your Horse.
  • Prestige (Experience) System with Skill points on Level Up.
  • Auction system for buying and selling Horses.

Technology

The Hippodrome is a Open-source game and the code can be found at: https://github.com/adrianosimoes/hippodrome It's a HTML5 game that uses Angular 6.0 as frontend technology and it saves the game data in the cookies, so it doesn't need any backend architecture so far.
